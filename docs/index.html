📐 Math Reference

RTP (Return to Player)
total_win ÷ total_wagered
Decomposed into Base + Bonus + H&S. Target ≈ 95% — house edge = 5%.
Volatility Index
√(variance) ÷ bet
Variance computed via Welford's online algorithm — numerically stable at 10M+ spins.
Hit Rate
spins_with_payout > 0 ÷ total_spins
Elevated by cascades — a single bet can count as a hit across multiple cascade levels.
Win Buckets
0× / 0–1× / 1–5× / 5–15× / 15–50× / 50×+
Log-scale distribution. Regulatory jurisdictions often mandate maximum bucket caps.
Trigger Frequency
total_spins ÷ trigger_count
Reported as "1 in X" for Free Spins, H&S, and Grand Jackpot separately.
